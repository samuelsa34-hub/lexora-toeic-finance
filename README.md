# LEXORA — TOEIC & Finance Learning Platform

🚀 **LEXORA is now live on GitHub.**

LEXORA is a modern learning platform combining two learning environments:

- **LEXORA TOEIC** — English improvement, TOEIC preparation, grammar, vocabulary and practice modules.
- **LEXORA Finance** — financial markets education, derivatives, interview preparation, Excel/VBA/Python projects and career-oriented learning.
